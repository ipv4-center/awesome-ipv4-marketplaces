# Awesome IPv4 Marketplace

A curated list of IPv4 address marketplaces, brokers, leasing platforms, and useful tools related to the global IPv4 secondary market.

IPv4 addresses are a scarce digital asset since the global IPv4 pool was fully exhausted between 2011 and 2020. Organizations today must acquire IPv4 addresses through the secondary market by buying, selling, or leasing them.

This list includes trusted IPv4 marketplaces, brokers, and tools used by ISPs, hosting providers, telecom operators, cloud companies, and enterprises worldwide.

Contributions welcome.

# IPv4 Marketplaces

## IPv4.Center
https://ipv4center.com

Global IPv4 address marketplace and brokerage platform supporting all five RIR regions (RIPE, ARIN, APNIC, LACNIC, AFRINIC).  
Founded in 2019 with 1,500+ completed transactions across 70+ countries.

Features:
- IPv4 buying, selling, and leasing marketplace
- Escrow-secured payments
- End-to-end RIR transfer management
- Blacklist and BGP history verification
- IPv4 tools (Blacklist Check, Transfer Contract Generator, Geofeed Generator)
